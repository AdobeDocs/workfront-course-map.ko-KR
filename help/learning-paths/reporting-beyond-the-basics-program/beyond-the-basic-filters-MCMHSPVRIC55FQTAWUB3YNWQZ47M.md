# 기본 사항 이후의 보고

Workfront One 학습 경로의 기본 사항을 벗어난 보고는 로 마이그레이션되었습니다. [중간 필터 만들기](https://experienceleague.adobe.com/?recommended=Workfront-U-1-2022.2.reporting){target="_blank"} Experience League 과정.

새로운 Experience League 과정은 사전 학습 경로와 동일한 학습 목표를 다루고, 가능한 한 최적화하여 포괄적인 학습 경험을 제공합니다.  Experience League에서 Workfront 교육 과정, 설명서, 튜토리얼 및 커뮤니티의 전체 라이브러리를 살펴보십시오.
